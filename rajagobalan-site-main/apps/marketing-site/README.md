# Marketing site (rajagobalan.com)

**Phase 1:** The public marketing site is built from the **repository root** with Vite (`vite.config.js`, `index.html`, `blogs.html`, `public/`, etc.). This folder exists as the **canonical home** for that app in the `apps/` layout.

**Future:** Optionally move all Vite inputs and `public/` into `apps/marketing-site/` and set `root` in `vite.config.js`—no change to Firebase Hosting URL behavior.
